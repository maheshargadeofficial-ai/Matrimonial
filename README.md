# Saksham Saathi - Matrimonial App for Persons with Disabilities

## 🎯 Play Store Ready Android App

India's first inclusive matrimonial platform for persons with disabilities.

### ✨ Features
- Complete Authentication System
- Profile Management with Photo Upload
- Browse & Search Profiles
- Interest System (Send/Receive/Accept/Reject)
- Matches Page
- Real-time Messaging System
- Multi-language Support (Hindi/English)
- Accessibility Features (High Contrast, Font Size)

### 📱 Build APK

APK automatically builds when you push code to GitHub. Download from Actions tab.

### 🚀 Tech Stack
- Frontend: React + Capacitor
- Backend: FastAPI + MongoDB
- Mobile: Capacitor Android

---

## 📥 APK Download Kaise Karein

1. Go to **Actions** tab (upar GitHub pe)
2. Latest workflow run pe click karein
3. Neeche **Artifacts** section mein **saksham-saathi-debug-apk** download karein
4. ZIP extract karke APK file milega
5. APK ko mobile pe install karein

---

## 🏪 Play Store Upload Steps

1. **Google Play Console** pe jao (play.google.com/console)
2. Create new app
3. App details bharein:
   - App name: Saksham Saathi
   - Category: Lifestyle
   - Language: Hindi/English
4. Upload APK in **Release** section
5. Screenshots, description add karein
6. Submit for review
